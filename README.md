# SkilledUp Frontends

This repository contains two Vite + React projects:
- `frontend/` (student website)
- `admin-frontend/` (admin panel)

## Production configuration
- Student frontend: set `frontend/.env.production` using:
  - `frontend/.env.production.example`
- Admin frontend: set `admin-frontend/.env.production` using:
  - `admin-frontend/.env.production.example`

## Build
Student:
- `cd frontend`
- `npm install`
- `npm run build`

Admin:
- `cd admin-frontend`
- `npm install`
- `npm run build`
